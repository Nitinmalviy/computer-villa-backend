# Ghumakkad Backend

This is the backend server for the Ghumakkad project.

## Setup

1. Clone the repository
2. Install dependencies:
```bash
npm install
```
3. Create a `.env` file with required environment variables
4. Run the development server:
```bash
npm run dev
```

## Technologies Used

- Node.js
- Express
- TypeScript
